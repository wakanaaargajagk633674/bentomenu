API費用記録機能・公式情報調査

アクセス日: 2026-07-14

この機能が表示する金額は、APIの使用量と公開単価から求める概算です。OpenAIの請求・利用明細を確定額として扱います。

1. OpenAI API Pricing

発行者
OpenAI
URL
https://developers.openai.com/api/docs/pricing
資料種別
公式料金表（一次資料）
主要な根拠
gpt-5.5短コンテキストの100万トークン当たり単価は、Flexが入力 $2.50、キャッシュ入力 $0.25、出力 $15.00、Standardが入力 $5.00、キャッシュ入力 $0.50、出力 $30.00、Priorityが入力 $12.50、キャッシュ入力 $1.25、出力 $75.00。GPT Image 2 Standardのトークン単価は画像入力 $8、テキスト入力 $5、出力 $30。
プロジェクトへの反映
Responses APIが返す入力・キャッシュ入力・出力トークンを分け、API応答の実処理tierに対応する単価でUSD費用を計算する。画像APIが使用量を返す場合にも同ページの単価を使う。
確信度
高
制約
料金は将来変更される可能性があるため、実装時点のスナップショット。長コンテキストや別モデルを設定した場合は単価が異なる。

2. Image generation — Calculating costs

発行者
OpenAI
URL
https://developers.openai.com/api/docs/guides/image-generation#calculating-costs
資料種別
公式APIガイド・料金計算例（一次資料）
主要な根拠
GPT Image 2の1024×1024、medium品質は画像1枚当たり $0.053 の見積。入力トークン費用は別途発生し得る。
プロジェクトへの反映
Image APIレスポンスから使用量を取得できない場合の代替値として、1生成 $0.053 を記録し「公式見積で計算」と表示する。
確信度
高
制約
1枚の見積値であり、プロンプトなどの入力費用を完全には表さない可能性がある。APIが使用量を返す場合はトークン計算を優先する。

4. Responses API reference

発行者
OpenAI
URL
https://developers.openai.com/api/reference/resources/responses/methods/create
資料種別
公式APIリファレンス（一次資料）
主要な根拠
Responses APIのレスポンス使用量に入力・出力トークンと入力詳細が含まれる。指定したservice tierと実処理tierが異なる場合があり、レスポンスには実際に使われた値が返る。
プロジェクトへの反映
候補選定と詳細生成の成功レスポンスごとに response.usage を読み、キャッシュ入力を通常入力と分離し、response.service_tierに応じた単価で記録する。
確信度
高
制約
SDK型ではusageが省略可能なため、返却されなかった呼び出しは費用記録を作らず生成処理を継続する。
