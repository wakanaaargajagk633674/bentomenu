Key evidence
正式モデルIDはgpt-image-2。Image Generation endpointに対応し、高忠実度入力を扱う最新画像モデル。Free tierは非対応。
Influence
Vercel環境変数OPENAI_IMAGE_MODELとサーバー側フォールバックをgpt-image-2に固定した。
Confidence
高。モデル提供元の公式仕様。
Limitations
生成結果のレシピ一致や同一プロンプトでのピクセル再現を保証する資料ではない。組織認証・利用枠は契約状態に依存。

公式API実装ガイド（一次資料）
Key evidence
openai.images.generate、base64画像応答、1024×1024、quality、WebP/JPEG、圧縮指定を確認。複雑なプロンプトは最大約2分を要し得る。GPT Image 2は透明背景非対応。1024正方形mediumは掲載価格上1枚約US$0.053。
Influence
画像専用Route、240秒上流タイムアウト、medium・1024正方形・WebP 82%、opaque背景、候補単位の段階生成を採用した。
Confidence
高。提供元の公式ガイド。
Limitations
価格、上限、機能は更新され得る。通信量・Vercel実行時間・為替・再試行費用は別途管理が必要。
