# 共通設計原則

## 1. 一流料理人に共通する思考

- **素材起点**: 旬、産地、個体差、熟度を見てから技法を決める。技法を見せるために素材を従わせない。
- **制約起点**: 価格、設備、人数、時間、容器は創造性を狭める障害ではなく、料理の輪郭を作る条件として扱う。
- **記憶起点**: 郷土、家庭、旅、季節の記憶を核にし、客が理解できる形へ翻訳する。物語は味を補強する場合だけ使う。
- **編集**: 足すより削る。各要素に「主役を強める」「対比を作る」「口をリセットする」「季節を示す」の役割を持たせる。
- **流れ**: 一皿だけでなく献立全体の強度を見る。軽→重、淡→濃だけに固定せず、酸味や温度でリズムを作る。

## 2. 味・香り・食感・温度

### 味の地図

主味を1つ、支持味を1～2つ、輪郭を作る味を1つ決める。

- 塩味: 味の輪郭。濃度ではなく均一性と入るタイミングを管理する。
- 酸味: 脂を切り、香りを立て、後味を短くする。酢だけでなく柑橘、乳酸発酵、漬物、トマトも使う。
- 甘味: 丸みと持続性。砂糖だけでなく炒め玉ねぎ、みりん、果物、穀物由来を区別する。
- 苦味: 奥行きと季節感。焦げ、山菜、葉物、柑橘皮を制御して使う。
- うま味: 厚み。発酵、乾物、熟成、出汁、フォンを重ねるが、重く濁らせない。
- 辛味・香味: 注意を戻すアクセント。全品を辛くせず、強弱と持続時間を変える。
- 脂肪: 香りの媒体と満足感。酸、薬味、苦味、温度差で均衡させる。

### 香りの三段階

1. 提供前に届く香り（炙り、湯気、柑橘、胡麻油）
2. 口に入れた瞬間の香り（香味野菜、スパイス、酒）
3. 咀嚼後の余韻（発酵、燻製、焦げ、木の芽）

香りを全部ソースに混ぜず、仕上げ油、薬味、粉、炙りで時間差を作る。

### 食感と温度

各料理に最低2つの食感を設ける。ただし衣やナッツを安易に足さず、切り方・火入れ・水分制御で作る。献立には温／冷、汁／乾、柔／脆、滑／粒の対比を置く。温度は味覚強度と香りの放出を変えるため、提供時点をレシピに含める。

## 3. レシピ開発

### レシピ骨格

- 料理の目的と合格像
- 正味重量と出来高
- 材料を機能別に分類（主材、塩味、酸味、甘味、うま味、脂肪、香り、食感、色）
- 前処理、切り方、投入順
- 中心温度／油温／煮詰め濃度／休ませ時間
- 仕上げと盛付け
- 保存、再加熱、提供限界

家庭の「適量」「焼き色がつくまで」を業務再現するときは、重量・温度・時間・見た目の指標へ翻訳する。試作では塩分、糖、酸の数値だけに依存せず、温度と提供形態を揃えたブラインド比較を行う。

### 試作ログ

仮説 → 変更変数 → 結果 → 次の一手を記録する。一度に複数を変えない。直後、15分後、弁当なら2～4時間後を評価し、香り、離水、衣、肉の硬化、色移りも記録する。

## 4. 色彩と盛り付け

- 主役の位置を最初に決め、視線の入口を1つ作る。
- 食材色は赤、黄、緑、白、黒／褐色を点検するが、全色を均等に置かない。主色60%、補助色30%、差し色10%を目安に調整する。
- 茶色い料理には、食べられる酸味・香味の差し色を使う。目的のないパセリは置かない。
- 高さは安定性と食べやすさを壊さない範囲。大きいものを奥、小さいものを手前にすると要素が見えやすい。
- 同じ形を並べると秩序、奇数と非対称は動き、余白は主役の強調になる。
- ソースは接着、味の順序、視線誘導の役割を持たせる。皿の模様として無意味に引かない。
- 器・容器、照明、蓋を開けた瞬間、箸やスプーンの動線まで含めて盛付けとする。

## 5. 創造性と文化

「伝統の核＋新しいレンズ」で設計する。

- 核: 味型、発酵調味料、基礎技法、食べる順序、地域食材のいずれか
- レンズ: 別の火入れ、形、温度、提供動作、現代的な軽さ、地域の別食材

融合料理では双方の文化を説明できない組合せを避ける。名前だけ借りない。まず基準となる伝統版を試作し、何を保持し何を変更したかを言語化する。
