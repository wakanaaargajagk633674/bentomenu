# 弁当・居酒屋への実装

## 目次

1. 弁当
2. 居酒屋
3. 共通の現場設計

## 1. 弁当

### 献立構造

- 主食、主菜、副菜2～4品、口直しを基本骨格とし、目的に応じて増減する。
- 主菜は満足の中心、副菜は色埋めでなく、酸味、苦味、食感、栄養、季節の役割を分担する。
- 同じ調味（醤油甘辛など）、同じ食材、同じ加熱法を重ねない。
- 冷めると塩味・甘味・脂の感じ方、肉やでんぷんの食感が変わる。必ず想定喫食温度で評価する。
- 汁気を切る、煮詰める、仕切る。揚げ物は蒸気を逃がし、ソースを別添えまたは接触面を限定する。
- 蓋を開けた瞬間に主役が見え、隙間なく安定し、箸で一品ずつ取れるように詰める。

### 色

容器を含めて赤・黄・緑・白・黒／褐色を確認する。ご飯の白と海苔・胡麻の黒を土台に、卵・南瓜の黄、葉物の緑、梅・人参・赤パプリカの赤を使える。ただし毎回同じ食材で五色を埋めない。紫、橙、淡緑など季節色へ展開する。

### 食品安全（絶対条件）

- 「つけない・増やさない・やっつける」を守る。
- 肉、魚、卵は中心まで十分に加熱する。半熟卵を常温弁当に使わない。
- 温かいまま蓋をせず、各品を速やかに冷ましてから清潔な器具で詰める。
- 水分を切り、生野菜・果物は洗浄後に水気を除き、必要なら別容器にする。
- 前日品は適切に冷蔵し、使用時に十分再加熱して再度速やかに冷却する。
- 長時間持ち歩く場合は保冷剤・保冷バッグを使い、早めに食べる前提を明示する。
- 商用提供ではHACCPに沿う衛生管理、自治体の営業許可・表示・アレルゲン要件を別途確認する。

## 2. 居酒屋

### メニュー構造

- 即出し、冷菜、温菜、焼き、揚げ、煮物、主食、甘味を配置し、厨房の火口・フライヤー負荷を分散する。
- 名物1～3品を中心に、利益を作る定番、季節の来店理由、酒をつなぐ小皿を置く。
- 一皿の味だけでなく注文の連続を設計する。最初は香りと酸、途中は焼き・揚げ・うま味、終盤は汁や主食で着地させる。
- 日本酒には出汁・発酵・塩味、ビールには香ばしさ・脂・辛味、ワインには酸・香草・ソースなどを出発点にするが、実際に試飲して決める。
- 共有しやすい個数、切り方、温度保持、取り皿での崩れ方を確認する。

### 独自性

「名物」は珍しい食材ではなく、記憶できる香り、提供動作、食感、地域の物語、優れた価格価値の組合せで作る。メニュー名は主材料＋技法＋決め手が伝わるようにし、説明しすぎない。

## 3. 共通の現場設計

- 食材の交差利用は廃棄削減になるが、全品が同じ味にならないよう技法と香りを変える。
- 仕込み一覧、保存限界、再加熱方法、盛付け秒数、ピーク時の同時注文を確認する。
- 原価率だけでなく、作業時間、歩留まり、廃棄、設備占有、売れ残りを含む実質原価で判断する。
- レシピは1人前と仕込み単位の両方を持ち、倍率変更で香辛料、塩、増粘剤を単純比例させない。
